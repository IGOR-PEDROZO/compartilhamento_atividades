Sobre o Projeto

Projeto executado por Igor José da Cunha Pedrozo. Consiste em uma atividade de extensão da faculdade Estácio. Esse projeto foi executado em conjunto com a pedagoga Any Caroline Fernandes Guimarães.

Esse projeto é uma aplicação simples desenvolvida com Flask, destinada a pedagogos. A ideia é compartilhar informações e atividades, facilitando a comunicação e a troca de recursos entre educadores.

Características do Projeto

  • Uma página inicial de apresentações e com links para outras seções.
  • Uma página que explica o objetivo do projeto.
  • Uma página para um calendário informando as atividades da semana.
